DESCRIPTION OF PROJECT
The Campus Bus Tracking Application (CBTA) is a database-based system designed to
manage and monitor university transportation services efficiently. The system provides a
centralized platform for managing students, drivers, administrators, buses, routes,
schedules, and real-time bus tracking information. 
The application allows administrators to manage users, assign drivers to buses, define
routes and stops, maintain bus schedules, and generate alerts. Students can access their
assigned bus information and track bus locations, estimated arrival times (ETA), and route
details. 
The main objective of this project is to improve campus transportation management by
reducing manual work, increasing efficiency, and providing accurate and timely bus tracking
information to students and transport administrators. The database system ensures data
consistency, security, and proper management of all transportation-related activities.
